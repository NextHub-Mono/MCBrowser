**MCBrowser** is a next-generation web browser that pursues minimalism and immersion.

Built on Electron, its black UI, stripped of unnecessary embellishments, makes your browsing more sophisticated.

[Official Website Here](https://mikancat10.github.io/MCBrowser/)

---

## ✨ Main Features

- 🌑 **White-Centric UI**: A modern and beautiful white mode design designed to enhance focus.

- 🕒 **Custom NewTab**: A minimalist configuration with only a clock and search bar. An experience where only what you need is there.

- ⚡ **Lightweight Performance**: Smooth operation that maximizes the power of Electron by eliminating unnecessary functions.

- 🖱️ **Enhanced Native Menu**: A uniquely customized right-click menu for speedy operation.

## 🛠 Setup (Installation)

If you are running or building in a development environment, please follow these steps:

### 1. Preparation
Ensure that Node.js is installed.

### 2. Installation
```bash
# Clone the repository
git clone [https://github.com/mikancat10/MCBrowser.git](https://github.com/mikancat10/MCBrowser.git)
# Navigate to the directory
cd MCBrowser
# Install dependencies
npm install
# Start
npm start
```
